# بطاقة تعريف نبات البفتة (Befta Plant Label)

## الوصف
هذا المستودع يحتوي على صفحة ويب بسيطة تعرض بطاقة تعريف نبات البفتة (Catharanthus roseus) بشكل منسق وواضح.

## كيفية الاستخدام
يمكنك فتح ملف `index.html` مباشرة في أي متصفح لمشاهدة البطاقة.  
كما يمكن نشر الصفحة باستخدام GitHub Pages للوصول إليها عبر الإنترنت.

## الملفات الموجودة
- `index.html`: صفحة الويب التي تحتوي على بطاقة تعريف النبات.

## المتطلبات
- متصفح ويب حديث (مثل Google Chrome، Firefox، Edge).

## كيفية النشر على GitHub Pages
1. تأكد من وجود ملف `index.html` في الفرع الرئيسي (`main` أو `master`).
2. في صفحة المستودع على GitHub، اذهب إلى "Settings" > "Pages".
3. اختر الفرع `main` كمصدر ثم اضغط "Save".
4. ستظهر لك رابط الموقع الذي يمكنك مشاركته لعرض البطاقة.

## المساهمة
مرحب بأي مساهمة لتحسين البطاقة أو إضافة نباتات أخرى.
